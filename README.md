DeviceHardwareInfo
==================
Common method to retrieve the iOS device being used. (Updated for iPhone 5c and iPhone 5s)


HowToUseIt
==================
NSLog(@"Device Model is = %@",[DeviceHardwareInfo deviceModelString]);
