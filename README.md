DeviceHardwareInfo
==================
Common method to retrieve the iOS device being used. (Updated for iPhone 6/ 6 Plus and iPad Air 2)


HowToUseIt
==================
NSLog(@"Device Model is = %@",[DeviceHardwareInfo deviceModelString]);
